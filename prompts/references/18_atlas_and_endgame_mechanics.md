# 18 — Atlas and endgame mechanics

This doc is conceptual — for **current modifier values, scarab effects, tablet specifics, or pinnacle drop tables**, fetch the wiki or patch notes. The map below is for *navigation*, not citation.

Distinct from "league mechanics in the campaign" — this doc is about the **mapping endgame** and pinnacle progression.

## How endgame works (both games)

The endgame loop has the same shape in both games even though the implementation differs:

1. **Reach the endgame** — campaign completion (PoE1: Acts 1-10 + Kitava; PoE2: 6 Acts at full release, 4 Acts + 3 interludes in EA).
2. **Acquire maps / waystones** — drop / craft them; quality and tier scale with progression.
3. **Apply atlas tree + map modifiers** — atlas passive points, scarabs (PoE1), tablets (PoE1 Settlers / PoE2), waystone affixes.
4. **Run maps** to gain XP, currency, league-mechanic resources, and pinnacle-access fragments.
5. **Engage pinnacle bosses** — top-tier targets with build-defining drops.
6. **Optimise farming strategy** per session — what mechanic, what tier, what tree allocation.

The endgame is a **strategy-selection game**, not just a combat game. Knowing which mechanic to engage for which currency is half the loop.

## PoE1 endgame

### Atlas Passive Tree

- ~120-180 allocatable points (varies by patch).
- Sub-trees per league mechanic: Beast / Ambush / Harbinger / Heist / Essence / Strongbox / Bestiary / Legion / Breach / Delirium / Metamorph / Eater of Worlds / Searing Exarch / Maps / Bosses / etc.
- Voidstones (4 total) raise tier baseline from T11 → T16.
- **Conceptual purpose**: specialise per session. A mapper wants different points than a boss-farmer.
- **Reset**: Orb of Unmaking refunds points cheaply.
- **Strategy artifact**: the atlas tree is the most important *strategic* artifact for PoE1 — more than any single piece of gear.

### Sextants / Tablets / Towers (Settlers 3.25+)

- **Pre-3.25**: sextants applied to atlas-region watchstones.
- **3.25+ Settlers**: tablets applied to towers within atlas regions. Towers radiate effects to maps within range.
- **Conceptual purpose**: layer high-value modifiers (more rare monsters, more currency, league-mechanic spawn-rate boost) onto a region for a session of farming.
- **Strategy artifact**: tower-tablet-radius layouts define farming throughput.

### Maven and the Atlas Memory Game

- Maven witnesses pinnacle bosses; collect 10 witnesses for The Maven encounter.
- Memory Game phase requires concentration — failure means no rewards.
- Uber Maven gated by Writ of Maven craft.
- **Conceptual purpose**: pinnacle access (Writs, Awakened gem upgrades, build-defining uniques like The Eternity Shroud).
- **Build pressure**: 10k+ EHP, fluent boss-fight execution.

### Sanctum (3.20 league → core)

- Repeatable run mechanic: enter Sanctum, navigate rooms, collect Resolve / Aureus.
- Relics modify run state; equippable across runs.
- Drops chase uniques: Original Sin, Sanctum-exclusive uniques.
- **Conceptual purpose**: SSF-friendly currency / unique source. Build-friendly because you can re-roll runs.
- **Build pressure**: dodge i-frame fluency, single-target burst for end-floor bosses.

### Heist (3.12 league → core)

- Rogue's Markers gate contracts (single-area heists) and blueprints (multi-area heists).
- Heist-specific Replica unique pool.
- League-mechanic loot only available via Heist (e.g., Replica Soul Tether).
- **Conceptual purpose**: targeted unique acquisition + currency from contract gem rewards.
- **Build pressure**: stealth mechanics, alarm-management, time-pressure fights.

### Delve (3.4 league → core)

- Infinite linear depth; Sulphite-gated.
- Niko's mechanic; depth scales monster level.
- Drops: Cluster Jewels, Fossils, Aul's Uprising-style depth-gated uniques.
- **Conceptual purpose**: infinite scaling content for unique-hunting + Cluster Jewel farming.
- **Build pressure**: ground-degen mitigation, sustained DPS for darkness fights.

### Ritual (3.13 league → core)

- Ritual altars in maps; defer rewards across multiple altars per map.
- Tribute-based reroll economy.
- Drops: ritual-exclusive currencies (Sacred Crystal, Polished Sextant pre-3.25), uniques.
- **Conceptual purpose**: deferred reward scaling — bank rewards across maps for big payoffs.
- **Build pressure**: contested encounters, multi-mob clear.

### Expedition (3.15 league → core)

- Logbook explorer faction grind: Druids of the Broken Circle, Order of the Chayula, Knights of the Sun, Black Scythe Mercenaries.
- Faction-specific reward currencies (Exotic Coinage, Astragali).
- Drops: faction uniques, Logbook chase items.
- **Conceptual purpose**: faction-specialised farming with Logbook randomness.
- **Build pressure**: AoE clear + ranged sustain for explosive encounters.

### Settlers (3.25 league → core)

- Town-building economy meta-mechanic.
- Worker dispatch, crops, ports, currency exchange via dock trade.
- Currency Exchange (within Settlers) is the most reliable bulk-conversion tool in PoE1.
- **Conceptual purpose**: passive currency economy + bulk-trade infrastructure.
- **Build pressure**: minimal — town side-channel does most of the work.

### Other endgame mechanics

- **Breach**: portal mechanic in maps, splinter shards → Breachstones → Domain bosses.
- **Legion**: timed army-summon, pre-3.x style dense pack farming.
- **Metamorph**: corpse-assembly fight, drops Catalysts.
- **Bestiary**: capture beasts, unlock crafting recipes (Imprints, FPA dupes, Beast crafts).
- **Strongbox**: surprise mob + currency dump.
- **Harbinger / Beyond / Abyss / Ambush**: lighter atlas-mechanics, often atlas-tree-tagged.

### Pinnacle bosses (PoE1)

- **Sirus** — final boss of the Conqueror line. Deathless practice = uber.
- **Maven** — atlas memory + 10-witness pinnacle. Uber via Writ.
- **Searing Exarch / Eater of Worlds** — 3.17 split pinnacle pair. Uber via The Crux / Infinite Hunger.
- **Shaper / Elder** — pre-3.17 pinnacle line, accessed via legacy fragments.
- **Catarina (Mastermind of Discord)** — Syndicate endgame fight.
- **The Black Star / The Brine King** — Atlas-region-specific bosses (now folded into Conquerors).
- **Build pressure**: 12-15k EHP, 5-20M DPS depending on tier.

## PoE2 endgame

### Atlas (current as of 0.4)

- Infinite-atlas hex map, expanding outward from spawn point.
- Each hex is a Waystone-tier map.
- Towers within hex regions apply Tablets (similar to PoE1 Settlers).
- Citadels are endgame zones with pinnacle access.
- Corrupted Nexus is the atlas-points pinch-point.

### Waystones

- Equipped to start a map, similar to PoE1 maps.
- Tier 1-15 (or per-version current cap).
- Affixes roll on Waystones (mob density, modifier intensity).
- Quality scales drop rates within the map.

### Towers

- Hex-grid points on the atlas.
- Apply Tablets that radiate effects (similar to PoE1 Settlers tablet/tower system).
- Tablet effects multiply across all maps within radius.
- **Strategy artifact**: tower-tablet network defines farming throughput.

### Tablets

- Currency-style item; equipped to towers.
- Effects: more mob density, more pack size, more rare modifiers, league-mechanic spawn-rate.
- Drop / craft system; Currency Exchange (PoE2-equivalent of Settlers) handles bulk.

### Citadels

- Endgame zones beyond regular maps.
- Multiple types: Iron Citadel, Stone Citadel, etc.
- Drop pinnacle-access keys.
- **Build pressure**: high — pinnacle-tier difficulty.

### Corrupted Nexus

- Atlas-points pinch-point.
- 30 atlas points + 5 unique-map = 10 atlas points = 40 core, then league-mechanic sub-trees.
- Conceptual purpose: gated progression for atlas-tree investment.

### Pinnacle bosses (PoE2 0.4)

- **Arbiter of Ash** — fire-pinnacle.
- **King in the Mists** — ice-pinnacle.
- **Sirus-equivalent** — name varies by version.
- Each pinnacle drops ascendant build-defining uniques.

## Strategy heuristics

When the agent helps a player optimise endgame:

1. **What's the goal** — currency / unique / boss kill / XP / atlas points?
2. **What mechanic outputs that goal best** — cross-reference `maxroll/poe1_currency.md` for PoE1 farming strategies.
3. **What atlas tree** supports that mechanic?
4. **What scarab / tablet / waystone affixes** stack on top?
5. **What build pressure** does the mechanic create — does the player's build handle it?

The mistake to avoid: treating "atlas farming" as one thing. It's actually a strategy menu, and choosing the right menu item beats over-investing in a wrong one.
